# Advent of Code 2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) written in Go.

## Running Tests

### Run tests for a specific day:
```bash
go test ./days/day01
```

### Run tests for all days:
```bash
go test ./days/...
```
